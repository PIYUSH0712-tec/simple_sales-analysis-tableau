#  Simple Sales Dashboard Design (Tableau Public)

##  Project Objective
Create a **basic interactive sales performance dashboard** to analyze sales trends by **Category and Region over Month (Month–Year)** using Tableau Public.

---

##  Dataset Used
- **Superstore_Sales.csv**  
  Columns: `Order Date`, `Region`, `Category`, `Sales`, `Profit`  
  Total Rows: `1195`

---

##  Tool Used
- **Tableau Public** (Free Cloud version)

---

##  Visualizations Included
1. **Line Chart** → Sales trend over Month-Year
2. **Bar Chart** → Sales performance by Region
3. **Pie chart** → Sales distribution by Category

---

##  Interactive Controls
- **Region & Category filters added** to dashboard layout (visible on web view)

---

##  Key Insights (Deliverable)

- East region shows the highest overall sales contribution.
- Sales peaked mid-year indicating strong seasonal performance.
- Office Supplies category holds the largest share in total sales.
- Profit trend follows sales closely, suggesting stable margins across regions.
